folder contains 4 functions:

1. main.m - main script
2. load_img.m - loads one data vector
3. load_set.m - loads set of data vectors
4. comp_eig.m - computes left and right singular vectors and also projection of the data on 'eigenspace'
5. min_morm.m -  function that does facial identification, based on min_norm detector

Each function includes comments, for more detailed information.

Folder also contains folder 'imgdata'. It contains training image set
